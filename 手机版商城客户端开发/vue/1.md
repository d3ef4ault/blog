## 安装环境


### 安装全局vue-cli脚手架
```
npm install --global @vue/cli
```

### 创建项目

```
vue create my-project-name
```
选择 `Manually select features`

选中（上下键+空格键） `Babel` `TypeScript` `Router` `Vuex` `Linter / Formatter`

输入三次 `y`

直接默认回车即可

### 安装 sass 支持

```
npm install -D sass-loader node-sass
```

### 运行

```
npm run serve
```

浏览器打开 http://localhost:8080
